# 二建

## 建设工程施工管理--施工进度管理

### 总进度目标和任务
#### 一
—项目总进度目标是项目决策阶段定义时确定的，项目管理的主要任务是在项目实施阶段对项目的目标进行控制。
建设项目总进度目标的控制是业主方项目管理的任务。
大型建设工程项目总进度目标论证的核心工作是通过编制总进度纲要论证总进度目标实现的可能性。
项目总进度不仅包含施工进度，还包含从设计前准备阶段工作进度到项目动用前的准备工作进度之间的所有进度。

#### 二
**总进度纲要的主要内容：**
> 纲要着重部署、规划、里程碑及条件。1. 项目实施的总体部署。  
2. 总进度规划。
3. 各子系统进度规划。
4. 确定里程碑事件的计划进度目标。
5. 总进度目标实现的条件和采取的措施等。

**总进度目标论证的工作步骤：**
> 工作步骤着重具体实施，如收集、分析、计划、调整等。  
> 先调查，再分析，再确定项目工作编码，再编制协调，最后调整报告。  
>   
1. 调查研究和收集资料。
2. 进行项目结构分析。
3. 进行进度计划系统的结构分析。
4. 确定项目的工作编码。
5. 编制各层）各级）进度计划。
6. 协调各层进度计划的关系和编制总进度计划。
7. 若所编制的总进度计划不符合项目的进度目标，则设法调整。
8. 若经过多次调整，进度目标无法实现，则报告项目决策者。

**项目进度计划系统：**
项目进度计划系统是项目进度控制的依据，在项目进展中随着编制计划所需资料逐步形成和逐步完善。
根据项目进度需要和用途的不同，业主方和项目各个参与方可以编制多个不同的项目进度计划系统。
根据计划的深度、功能、参与方和周期的不同而有4种进度计划系统。
进度控制是一个动态管理过程，包含：分析和论证；编制；定期跟踪并纠偏和调整。

业主方进度控制的任务是控制整个项目**实施阶段**的进度，包括：设计准备阶段、设计、施工、物资采购及项目动用前准备阶段等工作进度。

### 施工进度计划的类型及其作用
施工企业：施工生产计划，建设工程项目施工进度计划。
建设工程项目施工进度计划从计划的功能划分有：控制性、指导性和实施性。
控制性施工进度计划包括：施工总进度纲要或施工总进度计划，施工总进度规划。
控制性施工进度计划是施工进度控制的纲领性文件，是组织和指挥施工的依据。
**控制性施工进度计划主要作用：**
1. 论证施工总进度目标。
2. 分解施工总进度目标，确定里程碑事件的进度目标。
3. 编制实施性进度计划的依据。
4. 编制与该项目相关的其他各种进度计划的依据或参考依据。
5. 施工进度动态控制的依据。

实施性施工进度计划的编制须结合工程施工的具体条件，并以控制性施工进度计划所确定的里程碑事件的进度目标为依据。
月度施工计划和旬施工作业计划是用于直接组织施工作业的计划，它是实施性施工进度计划。
**实施性施工进度计划主要作用：**
1. 确定施工作业的具体安排。
2. 确定时间段资金、人工、施工机械、建筑材料的需求。

### 施工进度计划的编制方法
横道图进度计划：工作可按照时间先后、责任、项目对象、同类资源等进行排序。

**双代号网络计划基本概念：**
1. 双代号网络计划是通过箭线及其两端节点的编号表示工作的网络图。
2. 工作也称工序、活动、作业，泛指一项需要消耗人力、物力和时间的具体活动过程。
3. 双代号网络图中，每一条箭线表示一项工作，由箭尾节点开始，箭头节点结束，箭线上方标注工作名称，箭线下方标注完成工作所需的持续时间。
4. 虚箭线表虚设工作，不占用时间不消耗资源，有联系、区分和断路3个作用。
5. 节点：起点节点、中间节点、终点节点。
6. 逻辑关系：工艺关系（工艺过程、工作程序），组织关系。

时间参数的计算及关键工作
工作持续时间：一项工作从开始到完成的时间。
工期：计算工期（通过网络计划时间参数计算出的工期），要求工期（任务委托人要求的工期），计划工期（按要求工期和计算工期所确定的作为实施目标的工期）。
网络计划的计划工期：当要求工期已确定，则计划工期不得大于要求工期；当要求工期未规定，则计划工期等于计算工期。
关键工作：网络图中总时差最小的工作。当计划工期等于计算工期时，总时差为零的工作就是关键工作。
关键线路：自始至终全部由关键工作组成的线路或线路上总的工作持续时间最长的线路（在各条线路中，总时间最长的线路）。
在计算双代号网络计划的时间参数时，工作的最早开始时间应为其所有紧前工作的最早完成时间的最大值。

### 施工进度控制的任务和措施
根据施工任务委托合同对施工进度的要求控制施工工作进度。

**施工方进度控制的主要工作环节：**
1. 编制施工进度计划及相关资源需求计划。
2. 组织施工进度计划的实施。
3. 施工进度计划的检查与调整。
4. 施工进度计划的检查与调整（施工进度计划的调整内容有：工程量的调整、工作（工序）起止时间的调整、工作关系的调整、资源提供条件的调整、必要目标的调整）。

施工方进度控制措施：组织、管理、经济、技术。

**施工进度控制的组织措施：**
1. 健全组织体系。
2. 设置专门部门及专人负责。
3. 在任务分工表和管理职能分工表中标示进度控制的工作任务和相应的管理职能并落实。
4. 编制进度控制的工作流程。
5. 进行和进度控制会议相关的组织设计。

**施工进度控制的管理措施：**
1. 科学严谨的管理对管理思想和手段、承发包模式、风险与合同管理。
2. 用工程网络计划的方法编制进度计划。
3. 发承包模式的选择。
4. 风险分析及管理。（组织、管理、技术、合同、资源等风险）
5. 重视信息技术的应用。

**施工进度控制的经济措施：**
1. 编制资金、人力、物力资源需求计划。
2. 编制工程成本计划时，应考虑加快工程进度所需要的资金。

**施工进度控制的技术措施：**
1. 工程受阻时考虑设计技术的影响，可考虑变更。
2. 施工技术、施工方法、施工机械。

## 施工质量管理
工程建设活动中，参与建设各方依法对**质量**负责，施工单位对建设工程的**施工质量**负责。  

施工质量特性体现在：适用性、安全性、耐久性、可靠性、经济性及与环境的协调性。  

施工质量包含两方面：施工活动过程的质量；施工产品结果的质量。  

工程固有特性满足两方面：符合国家法律、行政法规和技术标准、规范的要求；满足建设单位的需要。  

质量要求是动态、发展和相对的。质量“好””坏“，以固有特性是否满足要求的程度来衡量。  

**施工质量验收合格应符合下列3个条件：**  

1. 依法施工。（国标及相关专业验收规范）
2. 按图施工。（工程勘察、设计文件）
3. 践约施工。（施工承包合同）  

**影响施工质量的主要因素：**  

1. 人Man。直接参与施工的决策者、管理者和作业者个人质量意识和质量活动能力。决定性因素。
2. 材料Material。物质基础、重要基础。分为工程材料和施工用料。
3. 机械Machine。重要物质基础，合理选择和正确使用时重要措施。
4. 方法Method。还包括BIM等信息技术。
5. 环境Environment。复杂多变和不确定性特点。施工现场自然环境；施工质量管理环境；施工作业环境。  

施工质量管理：在工程项目施工安装和竣工验收阶段，指挥和控制施工组织关于质量的相互协调的活动。包含为满足施工产品质量要求而开展的策划、组织、计划、实施、检查、监督和审核等所有管理活动的总和。
施工项目经理是工程项目**施工**的最高领导，应对工程项目施工负全责。  

施工质量控制是在明确质量方针的前提下，对施工方案和资源配置的计划、实施、检查和处置等进行控制，包括事前、事中和事后三个系统过程。  

施工质量控制的特点由建设项目的**工程特点**和**施工生产特点**决定的。  

**施工质量控制特点主要表现：**  

1. 因素多。包含自然条件、施工工艺技术、管理制度办法。
2. 难度大。建筑产品单件性、施工生产流动性、人员多、关系复杂导致的。
3. 过程控制要求高。工序衔接多、中间交接多、隐蔽工程多，质量具有一定过程性和隐蔽性。要强调过程控制。
4. 终检局限大。表面检查，难以发现质量隐患。  

**施工质量保证体系内容：**  

1. 质量目标。明确目标，符合总目标要求。以工程承包合同为依据，逐级分解为各级质量目标。
2. 质量计划。根据企业质量手册和项目质量目标编制。分为质量工作计划和质量成本计划——质量成本管理的基准。
3. 思想保证体系。全面质量管理思想。
4. 组织保证体系。成立QC小组，规章制度，任务、职责、权限，质量信息系统。
5. 工作保证体系。明确工作任务和建立工作制度，落实在施工准备、施工、竣工3个阶段。  

**PDCA：**  

1. 计划。首要环节。
2. 实施。包含两个环节——计划行动方案交底，按计划规定的方法及要求展开活动。
3. 检查。
4. 处理。以检查为基础，肯定成功经验。  

施工企业的质量管理体系是工程承包经营和质量管理的基础。
质量管理体系文件包括：质量手册、程序文件、质量计划和质量记录。

**事前质量控制：**
编制施工质量计划、明确质量目标、制定施工方案、设置质量管理点、落实质量责任。分析、预防。  

**事中质量控制：**  行为约束、监督控制、坚持质量标准控制工序工作质量控制点。  

**事后质量控制：**阻止、评价、认证和纠偏、发现改进受控。  

**施工质量控制的一般方法：**  

1. 质量文件审核。
2. 现场质量检查。  

**施工准备的质量控制（属于施工质量控制的事前控制工作）：**  

1. 施工质量控制的准备工作。工程项目划分（划分为单位工程、分部工程、分项工程和检验批）；技术准备的质量控制。
2. 现场施工准备的质量控制。工程定位和标高基准；施工平面布置。
3. 材料的质量控制。
4. 施工机械设备的质量控制。  

**工序施工质量控制：**  

1. 工序施工条件质量控制。各生产要素质量及环境条件质量——检查、测试、试验、跟踪监督。
2. 工序施工效果质量控制。事后质量控制，实测、统计分析、判断和纠偏。地基基础、主体结构、幕墙工程、钢结构及管道等需现场质检合格。  

施工技术参数包括：混凝土的外加剂掺量、水灰比、回填土的含水量、砌体砂浆饱满度、防水混凝土抗渗等级、钢筋混凝土结构的实体检测结果及混凝土冬季施工受冻临界强度等技术参数。  

**施工质量验收：**  包括施工过程的工程质量验收和施工竣工质量验收。  

**工程质量事故的概念：**  

1. 质量不合格。未满足某个规定的要求；未满足与预期或规定用途有关要求，称为质量缺陷。
2. 质量问题。不合格的必须返修、加固、报废，经济损失低于规定限额的称为质量问题。
3. 质量事故。违反法规和标准，结构安全、使用功能等质量缺陷，返修、加固、报废，经济损失高于规定限额。  

工程事故具有复杂性、严重性、可变性、多发性特点。  

**工程质量事故分类：**  
1. 事故损失分类。特别重大（30人以上死亡，或100人以上重伤，或1亿元以上直接经济损失）。重大（10人以上30人以下死亡，或50人以上100人以下重伤，或5000万以上1亿元以下直接经济损失）。较大（3人以上10人以下死亡，或10人以上50人以下重伤，或1000万以上50000万以下直接经济损失）。一般（3人以下死亡，或10人以下重伤，或100万以上100万以下直接经济损失）。  
2. 事故责任分类。指导、操作、自然灾害。  
3. 产生原因分类。技术原因；管理原因；社会经济原因；其他原因。  
施工的失误包括：缺乏基本业务知识，不具备上岗的技术资质，不懂装懂瞎指挥，胡乱施工盲干；施工管理混乱，施工组织、施工工艺技术措施不当；不按图纸施工，不遵守相关规范，违章作业；使用不合格的工程材料、半成品、构配件；忽视安全施工，发生安全事故等，所有这一切都可能引发施工质量事故。

**导致事施工质量事故发生的主要原因：**

1. 非法承包，偷工减料。  
2. 违背基本建设程序。  
3. 勘察设计的失误。  
4. 施工的失误。  
5. 自然条件的影响。  

**施工质量事故处理的依据：**  

1. 质量事故的实况资料。  
2. 有关的合同文件。  
3. 有关的技术文件和档案。  
4. 相关的建设法规。  

**施工质量事故的处理程序：**  

报告给建设单位负责人————1小时内报告给事故发生地县级以上人民政府住房和城乡建设部门。同时，施工项目有关负责人采取措施第一时间抢救人员财产。   
事故处理结束后，必须尽快向主管部门提交完整的事故处理报告，其内容包括：事故调查的原始资料、测试的数据；事故原因分析、论证；事故处理的依据；事故处理的方案及技术措施；实施质量处理中有关的数据，记录、资料；检查验收记录；事故处理的结论等。  

建设单位应将施工、设计、监理和建设单位各方分别签字的质量验收证明在验收后3天内报送工程质量监督机构备案。  

### 习题知识点

1. 项目施工质量保证体系要以工程承包合同为基本依据，逐级分解目标，以形成在合同环境下的各级质量目标。  
2. 项目施工质量保证体系应有可行的质量计划，质量计划应根据企业的质量手册和项目质量目标来编制。  
3. 在质量体系文件的构成中，阐明一个企业的质量政策、质量体系和质量实践的文件是质量手册，它是实施和保持质量体系过程中长期遵循的纲领性文件。  
4. 在施工质量控制的准备工作中，技术准备工作包括熟悉施工图纸，进行详细的设计交底和图纸会审，制订施工质量控制计划，设置质量控制点，明确关键部位的质量管理点等。  
5. 在施工过程的质量控制中，工序的质量控制是施工阶段质量控制的重点，必须以工序质量控制为基础和核心。  
6. 影响施工质量的主要因素包括：人、材料、机械、方法及环境5大方面。  
7. 工程项目的施工质量保证体系的内容主要包括项目施工质量目标、项目施工质量计划、思想保证体系、组织保证体系和工作保证体系5个方面。  
8. 质量管理体系文件由质量手册、程序文件、质量计划和质量记录等构成。  
9. 施工质量控制中应强调过程控制，故事中控制的关键是坚持质量标准，控制的重点是工序质量、工作质量和质量控制点的质量。  
10. 目测法采用的手段是“看、摸、敲、照”。实测法采用的手段是“靠、量、吊、套”。  
11. 施工质量控制的准备工作包括工程项目划分与编号、技术准备的质量控制两个环节。  
12. 分项工程应按主要工种、材料、施工工艺、设备类别等进行划分。  
13. 检验批可根据施工及质量控制和专业验收需要按楼层、施工段、变形缝等进行划分。  
14. 工程项目分部工程质量验收合格的基本条件包括所含分项工程验收合格，质量控制资料完整，观感质量验收符合要求，涉及安全和使用功能的分部工程检验结果符合要求。  
15. 监督机构按照监督方案对工程项目全过程施工情况进行不定期的检查，其中对基础和主体结构阶段的施工应每月安排监督检查。    

## 五、 施工执业健康安全与环境  
### 职业健康安全管理体系与环境管理体系  

《职业健康安全管理体系 要求》的构成要素包括方针、策划、实施与运行、检查和纠正措施、管理评审.  

《职业健康安全管理体系要求》的构成要素包括方针、策划、实施与运行、检查和纠正措施、管理评审。  

作业文件是指管理手册、程序文件以外的文件，一般包括作业指导书（操作规程）、管理规定、监测活动准则及程序文件引用的表格。  

在专项施工方案专家论证制度中，要求对涉及深基坑、地下暗挖工程、高大模板工程的专项施工方案，施工单位还应当组织专家进行论证、审查。  

第2类危险源主要体现在设备故障或缺陷（物的不安全状态）、人为失误（人的不安全行为）和管理缺陷等几个方面。  

第1类危险源控制方法：可以采取消除危险源、和隔离危险物质防护、应急救援等方法。  
第2类危险源的控制方法：应采取提高各类设施的可靠性以消除或减少故障、增加安全系数、设置安全监控系统、改善作业环境等方法。  

**施工安全隐患处理原则：**

1. 冗余安全度处理原则。多道防线。    
2. 单项隐患综合处理原则。“人、机、料、法、环”。    
3. 直接隐患与间接隐患并治原则。  
4. 预防与减灾并重处理原则。  
5. 重点处理原则。  
6. 动态处理原则。  

安全隐患主要来自人、物、管3个方面的不安全因素，其中，人是指人的不安全行为，人在心理、生理和能力等方面的不安全因素；物是指物的不安全状态，比如设备设施、施工现场环境等方面的缺陷；管是指管理上的不安全因素，即对人、物、工作的管理不当。  

### 施工安全生产管理  
### 生产安全事故应急预案和事故处理  

生产经营单位应当每年至少组织1次综合应急预案演练，每半年至少组织1次现场处置方案的演练。  

**施工生产安全事故的处理：**  
1. 事故的处理原则。“四不放过”原则①事故原因没有查清不放过；②责任人员没有受到处理不放过；③职工群众没有受到教育不放过；④防范措施没有落实不放过。  
2. 事故报告的要求。由施工单位事故报告要求和建设行政主管部门事故报告要求组成。
3. 事故报告的内容。实行施工总承包的建设工程，由总承包单位负责上报事故。当事故发生后出现新情况，以及事故发生之日起30日内伤亡人数发生变化的，则应当及时补报。  
4. 事故调查。事故调查报告应当附上有关证据材料，并由事故调查组成人员在事故调查报告上签名。  
5. 事故处理。建设主管部门应当按照有关法律法规的规定，对因降低安全生产条件导致事故发生的施工单位给予暂扣或者吊销安全生产许可证的处罚。  
6. 法律责任。有关地方政府或者有关部门故意拖延或者拒绝落实经批复的对事故责任人的处理意见的，由监察机关对有关责任人员依法给予处分。    

指导责任事故是指由于工程指导或领导失误而造成的质量事故，例如，片面追求施工进度，放松或不按质量标准进行控制和检验，降低质量标准等而造成的质量事故。

### 施工现场文明施工和环境保护要求  

**五牌一图：**工程概况牌、管理人员名单及监督电话牌、消防保卫（防火责任）牌、安全生产牌、文明施工牌和施工现场平面图。  

**环境保护原则：**  
1. 经济建设与环境保护协调发展的原则。  
2. 预防为主、防治结合、综合治理的原则。  
3. 依靠群众保护环境的原则。  
4. 环境经济责任制原则（污染者付费的原则）。  

建设工程施工由于受技术和经济条件限制，对环境的污染不能控制在规定范围内的，建设单位应与施工单位共同报请当地人民政府建设行政主管部门和环境保护行政主管部门批准。  

对施工场界内的污染防治属于职业健康安全问题，而对周围环境的污染防治是环境保护的问题。

固体废物处理的基本思想是：采取资源化、减量化和无害化的处理，对固体废物产生的全过程进行控制。

2. 安全检查制度中，安全检查的重点是检查“三违”和安全责任制的落实。

## 施工合同管理  

### 施工发承包模式  

### 施工合同与物资采购合同  

**进度控制的主要条款内容：**  

1. 合同进度计划。指承包人应按专用合同条款约定的内容和期限编制详细的施工进度计划和施工方案说明报送监理人批准，经监理人批准的施工进度计划，它是控制合同工程进度的依据。  
2. 开工日期与工期。工期按监理人发出的开工通知中载明的日期计算。  
3. 工期调整。  
4. 暂停施工。监理人；暂停施工56天。  

**质量控制的主要条款内容：**  

1. 承包人的质量管理。机构、人员、制度；质量保证措施文件报送监理人审批。    
2. 承包人的质量检查。质量检查和检验，并作详细记录；编制工程质量报表并报送监理人审查。    
3. 监理人的质量检查。  
4. 工程隐蔽部位覆盖前的检查。承包人未通知监理人到场检查，私自将工程隐蔽部位覆盖的，监理人有权指示承包人钻孔探测或揭开检查，由此增加的费用和（或）延误的工期由承包人承担。    
5. 消除不合格工程以及实验和检验等。  

发包人对工程质量有异议，已竣工验收或竣工未验收但实际投入使用的工程，其质量争议按该工程保修合同执行。

由于承包人原因造成某项工程或工程设备不能按原定目标使用而需要再次检查、检验和修复的，发包人有权要求承包人相应延长缺陷责任期，但缺陷责任期最长不超过2年。  

如果责任属于承包人原因造成的，应由承包人承担由此责任发生的的费用；如果属于发包人原因造成的，则由发包人承担由此责任发生的费用，并支付承包人合理利润。  

若分包人与发包人或工程师（监理人）发生直接工作联系，将被视为违约，并承担违约责任。  

分包工程合同价款采用的形式应与总包合同约定的方式一致。  

分包合同价款与总包合同相应部分价款无任何连带关系。  

当劳务分包人和工程承包人对劳务报酬结算价款发生争议时，按合同约定处理。  

对使用工程量清单计价的工程，宜采用单价合同，但并不排斥总价合同。  

成本加奖金合同适用于在招标时，当图纸、规范等准备不充分，不能据以确定合同价格，而仅能制定一个估算指标时采用。  

最大成本加费用合同是指在工程成本总价基础上加酬金费用的方式，即当设计深度达到可以报总价的深度，投标人报一个工程成本总价和一个固定的酬金（包括各项管理费、风险费和利润）。  

固定总价合同的价格计算是以图纸及规定、规范为基础，工程任务和内容明确，业主的要求和条件清楚，合同总价一次包死，固定不变，即不再因为环境的变化和工程量的增减而变化。在这类合同中承包商承担了全部的工程量和价格的风险。  

**合同跟踪的对象：**  

1. 承包的任务。施工质量、工程进度、工程数量、成本的变化。  
2. 承包人跟踪检查分包人或分包工程小组，协调、意见、建议或警告，总承包协调管理责任。
3. 业主等。  

合同变更是指合同成立至合同履行完毕这段时间由双方当事人依法对合同的内容所进行的修改，合同变更包括合同价款、工程内容、工程的数量、质量要求和标准、实施程序等的一切改变。   

经发包人同意，监理人可根据合同约定的变更程序向承包人作出变更指示，承包人应遵照执行。变更指示必须由监理人发出。没有监理人的变更指示，承包人不能擅自变更。  

索赔的依据主要包括合同文件；法律、法规；工程建设惯例。  

索赔证据的基本要求是：真实性、全面性、及时性、关联性和有效性。  

索赔成立，必须同时具备以下3个条件：①承包人按合同的规定提交索赔意向通知书和索赔报告；②索赔事件已造成了承包人工程项目成本增加和或直接工期损失；③索赔事件的责任不属于承包人的行为责任或风险责任。
